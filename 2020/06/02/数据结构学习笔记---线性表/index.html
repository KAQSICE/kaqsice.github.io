    线性表

    什么是线性表?
    • 线性表是指n个数据元素的有限序列(n >= 0)，n称作表长。
    线性表都有哪些特点?
    • 同一个线性表中元素具有相同特性
    • 相邻元素之间存在序偶关系
    • 除了第一个元素之外，其他每个元素都有且仅有一个直接前驱
    • 除了最后一个元素外，其他每个元素都有且仅有一个直接后继

    大概就是像一条线一样连在一起的结构

    线性表都有哪些类型呢?

    线性表按照存储结构的不同可以分为两种，使用顺序存储结构的称之为顺序表，使用链式存储结构的称之为链表

    顺序表

    即将线性表中的元素相继存放在一个连续的存储空间中。

    一般可以用数组来表示存储一个顺序表。

    特点

    线性表的顺序存储方式

    存取方式

    顺序存取和随机存取

    基本操作

    搜索

    顺序表的元素的地址是可以计算的，这样就可以实现随机访问，如

    LOC(ai) = LOC(a1) + (i - 1) * l

    l是一个数据元素所占空间大小

    如果搜索概率相等，顺序表搜索成功的平均比较次数为ACN=1+n2ACN=\frac {1+n}{2}ACN=21+n​，搜索不成功就比较n次。

    插入

    如果插入概率相等时，平均数据移动次数为AMN=n2AMN=\frac{n}{2}AMN=2n​。

    删除

    如果删除概率相等是，平均数据移动次数为AMN=n−12AMN=\frac{n-1}{2}AMN=2n−1​。

    总结

    把顺序表看作一个数组就好。

    链表

    链表是线性表的链接存储表示。

    特点

    存储单元可以不连续。

    存取方式

    顺序存取

    分类

    链表分为四类

    • 单链表
    • 循环链表
    • 静态链表
    • 双向链表

    单链表

    单链表存在一个数据域和一个指针域，数据域存放该结点的数据，指针域用来存放后继元素的地址。

    代码实现

    下面用C++来定义一个单链表SingleLinkedList

    基本操作

    插入(三种情况)

    • 在第一个结点前插入

      newNode->next = first;

      first = newNode;

    • 在链表中间插入

      newNode->next = p->next;

      p->next = newNode;

    • 在链表末尾插入

      newNode->next = p->next;

      p->next = newNode;

    删除

    删除结点p后面的元素

    q = p->next;

    p->next = q->next;

    带有头结点的单链表

    带有头结点的单链表的表头结点位于表的最前端，本身没有数据，只是用来标记表头。

    设计表头结点可以让链表操作的实现简化。

    基本操作

    插入

    q->next = p->next;

    p->next = q;

    删除

    q = p->next;

    p->next = q->next;

    delete q;

    前插法建立单链表

    前插法即先建立一个结点，每次读取数据将数据存放至结点的数据域里面，再将该结点插入链表前端，知道读入结束符为止。

    尾插法建立单链表

    尾插法每次将新结点加入到链表的表尾，引入了尾指针r，总是指向表的最后一个结点，新结点插在它后面。

    双向链表

    双向链表的结点中含有两个指针域，一个指向直接前驱，一个指向直接后继。

    代码实现

    基本操作

    插入

    q->prior = p;

    q->next = p->next;

    p->next = q;

    q->next->prior = q;

    删除

    p->next->prior = p->prior;

    p->prior->next = p->next;

    顺序表与链表的比较

    基于空间的比较

    存储分配方式

    • 顺序表的存储空间是静态分配的
    • 链表的存储空间是动态分配的

    存储密度

    (存储密度 = 结点数据本身的存储量 / 结点结构占用的存储量)

    • 顺序表的存储密度 = 1
    • 链表的存储密度 < 1

    基于时间的比较

    存取方式

    • 顺序表可以随机存取，也可以顺序存取
    • 链表只能顺序存取

    插入删除操作时移动元素个数

    • 顺序表平均移动一半元素
    • 链表只需要稍微修改一下指针即可
