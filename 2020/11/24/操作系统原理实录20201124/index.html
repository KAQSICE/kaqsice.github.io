操作系统原理实录20201124

存储管理(Storage Management)

挂载(Mount)

使用mount指令进行挂载

使用umount指令解除挂载

配额(Quota)

可以针对每个用户设置限额，有两个限额:

Soft limit --> Warning only

Hard limit --> Error

配额是针对文件系统的

常用命令

tar

-c create

-v verbose

-f file

-z gzip

-j bzip2

-x extract

dd(Disk Duplicate)

-if input file

-of output file

-bs block size

-count block count

磁盘(Disk)

磁盘臂调度算法(Disk Arm Scheduling Algorithms)有计算题

  • 先来先服务(FCFS)
  • 最短寻道时间优先(SSTF)
  • 电梯算法(扫描算法)(SCAN)
  • 循环电梯算法(C-SCAN)
